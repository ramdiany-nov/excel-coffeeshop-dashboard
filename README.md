## 📊 Project Overview
This project presents an end-to-end sales analysis using Microsoft Excel. The dashboard enables interactive filtering and highlights key business metrics including:
- Total sales over time  
- Sales by country  
- Top 5 customers  
- Performance by roast type  
- Product size analysis  
- Loyalty card impact  
The goal of this project is to demonstrate data analysis, dashboard design, and Excel-based business intelligence skills.

## 🛠 Tools & Features Used
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Slicers (Date, Roast Type, Size, Loyalty Card)  
- XLOOKUP  
- IFS  
- Data cleaning & transformation  

## 📈 Dashboard Components
### 1️⃣ Sales Timeline
Line chart showing total sales trends over multiple years segmented by:  
- Arabica  
- Excelsa  
- Liberica  
- Robusta  

### 2️⃣ Sales by Country
Bar chart comparing total revenue across:  
- United States  
- United Kingdom  
- Ireland  

### 3️⃣ Top 5 Customers
Horizontal bar chart ranking highest revenue customers.

### 4️⃣ Interactive Filters
Users can filter by:
- Order date (timeline slicer)
- Roast type
- Coffee size
- Loyalty card usage

## 📂 Dataset Information
The dataset includes 3 tables:
- orders
- customers
- products

## 🎯 Business Questions Answered

- Which country contributes most to total sales?
- Who are the top customers?

## 🚀 How to Use
- Download the Excel file.
- Open in Microsoft Excel (recommended: Excel 365 / 2019+).
- Use slicers to interactively explore sales performance.
- Adjust filters to generate dynamic insights.
